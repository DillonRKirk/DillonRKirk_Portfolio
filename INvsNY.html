Indiana vs. New York

This multivariate map is the micro to Waste per State Resident (2022)'s macro. It uses the same choropleth classes as Waste per State Resident (2022). However, this visualizes the total normalized waste (in tons) hosted in Indiana and New York landfills using six proportional symbol classes. Furthermore, each landfill is given a value-by-alpha quantity that symbolizes the percentage of methane flared by landfill.

I created the below map using ArcGIS Pro. I transformed the projected coordinated system into variations of USA Albers Equal Area Conic. Indiana was give a Standard Meridian of -86.2919129, a Standard Parallel of 13.167433488, and a second Standard Parallel of 53.068747088. For New York, I chose a Standard Parallel of -75.5962723, a first Standard Parallel of 14.161421142, and a Second Parallel of 57.074818542.

Then, I transformed the projected coordinate system into a Web Auxillary Mercator and uploaded it into ArcGIS Online. My argument is that, while we should encourage less waste per capita, we should also encourage flaring landfill methane into renewable energy. You can find my ArcGIS StoryMaps publication here.
